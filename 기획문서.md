# EWA 영어 학습 앱 기획문서

## 1. 프로젝트 개요

### 1.1 목표
한국 사용자들이 영문 도서를 읽으면서 동시에 한글 번역과 원문을 비교하고, TTS를 활용해 발음을 익히며, 어려운 단어는 즉시 확인할 수 있는 통합 영어 학습 플랫폼 제공.

### 1.2 핵심 가치
- **병렬 학습**: 원문과 번역을 동시에 보며 자연스러운 문맥 습득
- **멀티모달**: 읽기, 듣기, 쓰기를 통합한 학습
- **인터랙티브**: 단어 드래그로 즉시 의미 확인 및 학습 덱에 추가
- **오프라인 지원**: 네트워크 없이도 다운로드된 책 읽기 가능

---

## 2. 주요 기능

### 2.1 책 읽기 (Book Reader)
**목적**: 영문 도서를 한글 번역과 함께 읽기

**상세 기능**:
- 원문과 번역을 나란히 또는 상하로 표시
- TTS 버튼으로 현재 문장/단락 재생
- 번역 토글 기능 (번역 숨기기/보이기)
- 독서 진도 저장 (북마크, 마지막 읽은 위치)

**UI 요소**:
```
[← Back] [책제목] [⚙️ 설정]
─────────────────────────────────
영문: The quick brown fox...
한글: 빠른 갈색 여우가...
─────────────────────────────────
[🔊 TTS] [번역 숨기기] [북마크]
```

### 2.2 단어 학습 (Word Learning)
**목적**: 드래그한 단어의 의미를 즉시 확인하고 학습 목록에 추가

**상세 기능**:
- 단어 드래그/탭 → 팝업에 뜻 표시
  - 영문 정의
  - 한글 번역
  - 예문 (앱 내 출현 위치)
  - 발음 (IPA + 음성)
- "단어 추가" 클릭 → 개인 어휘장에 저장
- 여러 뜻 중 선택 가능 (Word sense disambiguation)

**팝업 UI**:
```
┌─────────────────────────┐
│ fox (noun)              │
│ [영문] a cunning animal │
│ [한글] 여우             │
│ [예문] The quick...     │
│ [🔊 발음]               │
│ [➕ 추가] [닫기]        │
└─────────────────────────┘
```

### 2.3 단어 복습 (Vocabulary Review)
**목적**: 스페이스드 리피티션으로 장기 기억 형성

**상세 기능**:
- 플래시카드 모드
- 선택지 문제 (4개 선택지)
- 듣기 테스트 (음성 → 선택)
- 진행률 및 복습 통계
- 복습 스케줄 (Day 1, 3, 7, 30일)

**복습 카드**:
```
┌──────────────────────┐
│ fox                  │
│ 다음 중 뜻이 맞나요? │
│ ○ 여우               │
│ ○ 늑대              │
│ ○ 곰                │
│ ○ 사슴              │
│ [정답] [다음]       │
└──────────────────────┘
```

### 2.4 통계 & 분석 (Analytics)
**목적**: 학습 진도 시각화

**상세 기능**:
- 독서량 (페이지/분)
- 습득 단어 수
- 복습 통과율
- 학습 스트릭 (연속 일수)
- 주간/월간 차트

## 3. 기술 스택

### 3.1 프론트엔드
```
Framework: React 18 + TypeScript
State Management: Zustand (경량, 빠른 상태 관리)
Styling: TailwindCSS + shadcn/ui
TTS: Web Audio API + TTS 서비스 (Google Cloud / Azure)
드래그: React-Dnd 또는 HTML5 Drag API
로컬 저장: IndexedDB (책, 단어장, 진도)
```

### 3.2 백엔드
```
Framework: Node.js (Express / Next.js API Routes)
Database: PostgreSQL (사용자, 책, 번역, 단어 관계)
Cache: Redis (인기 단어, TTS 캐시)
File Storage: AWS S3 / Cloudflare R2 (책 파일, TTS 음성)
Authentication: JWT + OAuth2 (구글, 카카오)
```

### 3.3 데이터 포맷
```
책: EPUB / PDF + JSON 메타데이터
번역: 문장 정렬 데이터 (Sentence alignment)
  {
    "bookId": "uuid",
    "sentences": [
      {
        "id": 1,
        "en": "The cat sat on the mat.",
        "ko": "고양이가 매트 위에 앉았다.",
        "startPos": 0,
        "endPos": 28
      }
    ]
  }
사전: JSON 또는 SQLite
  {
    "word": "fox",
    "pos": "noun",
    "definitions": [
      {
        "en": "a cunning animal",
        "ko": "여우",
        "examples": ["The quick brown fox..."]
      }
    ],
    "ipa": "/fɑːks/",
    "audioUrl": "..."
  }
```

## 4. 사용자 여정 (User Flow)

### 4.1 신규 사용자
```
1. 앱 설치
2. 회원가입 (이메일 / 소셜)
3. 레벨 선택 (초급 / 중급 / 고급)
4. 책 추천 피드
5. 책 선택 → 독서 시작
```

### 4.2 기존 사용자 - 읽기 플로우
```
1. 메인 화면 (진행 중인 책 목록)
2. 책 선택 → 마지막 위치에서 재개
3. 읽기 중:
   - 단어 드래그 → 정의 팝업
   - 팝업에서 "추가" → 어휘장 저장
   - TTS 재생
   - 번역 토글
4. 북마크 (중요 문구 저장)
5. 진도 저장 후 종료
```

### 4.3 복습 플로우
```
1. 탭: "어휘장"
2. 복습 필요 단어 필터링
3. 복습 모드 시작 (플래시카드 / 선택지 / 듣기)
4. 진행도 추적 및 스케줄 업데이트
5. 통계 확인
```

## 5. 데이터 모델

### 5.1 핵심 엔티티

**User**
```
- id: UUID
- email: String
- password: hashed
- name: String
- level: Enum (beginner, intermediate, advanced)
- createdAt: DateTime
- lastLoginAt: DateTime
```

**Book**
```
- id: UUID
- title: String
- author: String
- level: Enum
- coverUrl: String
- epubUrl / pdfUrl: String
- description: String
- totalPages: Int
- createdAt: DateTime
```

**Sentence**
```
- id: UUID
- bookId: UUID (FK)
- pageNumber: Int
- enText: String
- koText: String
- audioUrl: String (TTS 캐시)
- wordIds: UUID[] (FK to Word)
```

**Word**
```
- id: UUID
- word: String
- pos: String (noun, verb, adj...)
- definitions: JSON
  - en: String
  - ko: String
  - examples: String[]
- ipa: String
- audioUrl: String
```

**UserVocabulary**
```
- id: UUID
- userId: UUID (FK)
- wordId: UUID (FK)
- addedAt: DateTime
- nextReviewAt: DateTime
- reviewCount: Int
- correctCount: Int
- status: Enum (new, learning, mastered)
```

**ReadingProgress**
```
- id: UUID
- userId: UUID (FK)
- bookId: UUID (FK)
- currentPage: Int
- lastReadAt: DateTime
- totalTimeSpent: Int (seconds)
- bookmarks: JSON[]
```

- 커뮤니티 기능 (읽고 있는 책 공유, 토론)
